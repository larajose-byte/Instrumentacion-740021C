Unidad II — Dinámica de Procesos
// MODELADO · RESPUESTA DINÁMICA · ESTABILIDAD · ANÁLISIS EN FRECUENCIA
¿Por qué estudiar dinámica de procesos? Un controlador PID perfectamente sintonizado sobre un modelo equivocado del proceso generará inestabilidad o pobre desempeño. La dinámica de procesos proporciona las herramientas matemáticas para describir cómo responde un proceso al paso del tiempo ante cambios en sus entradas, y determinar con rigor si un sistema de control será estable o no. Es el puente entre la instrumentación (Unidad I) y el diseño avanzado de control.
FIG. 1 — Mapa conceptual Unidad II. La dinámica de procesos provee el modelo G(s) y los criterios de estabilidad necesarios para diseñar controladores PID y sistemas de control avanzado.
Objetivos de Aprendizaje
MODELADO — 5a
Modelado Matemático
Derivar modelos de procesos químicos usando balances de masa y energía. Obtener la función de transferencia G(s) mediante la transformada de Laplace.
  • Plantar y resolver ecuaciones diferenciales ordinarias (EDO)
  • Linealizar modelos no lineales alrededor del punto de operación
  • Obtener G(s) = Y(s)/U(s) para procesos típicos de Ing. Química
RESPUESTA — 5b
Análisis de Respuesta Dinámica
Predecir y calcular la respuesta temporal de sistemas de primer, segundo y alto orden ante entradas típicas: escalón, rampa, impulso y sinusoidal.
  • Sistemas de 1er orden: τ, Kp, tiempo de subida, estado estacionario
  • Sistemas de 2do orden: ζ, ωn, sobreimpulso, tiempo de asentamiento
  • Sistemas de alto orden: aproximaciones FOPDT, efectos de ceros y polos adicionales
ESTABILIDAD — 5c
Análisis de Estabilidad
Determinar si un sistema de control en lazo cerrado es estable sin necesidad de calcular las raíces del polinomio característico usando el criterio de Routh-Hurwitz.
  • Criterio de Routh-Hurwitz: condición necesaria y suficiente
  • Localización de polos en el plano s (semiplano izquierdo vs derecho)
  • Margen de ganancia Gm, margen de fase Pm y margen de módulo Mm

En la Unidad I identificamos experimentalmente el modelo FOPDT de un proceso: Kp, τ, θ. Instrumentación sin dinámica es medición sin comprensión.
Modelado Matemático de Procesos
// EDO · LINEALIZACIÓN · BALANCES DE MASA Y ENERGÍA · TRANSFORMADA DE LAPLACE
El modelado matemático es el proceso de traducir el comportamiento físico/químico de un proceso en ecuaciones que puedan analizarse y usarse para diseñar controladores. Un modelo útil no necesita ser perfecto: debe capturar la dinámica relevante en el rango de operación de interés.
Principios de Modelado
Balance de Masa (No Reactivo)
Acumulación = Entrada − Salida. Para un sistema con una sola corriente de entrada y una de salida en volumen de control V.
V·(dC/dt) = F_in·C_in − F_out·C_out
Balance de Energía (Tanque Agitado)
Acumulación de energía = Energía que entra − Energía que sale ± Calor intercambiado.
ρ·Cp·V·(dT/dt) = F·ρ·Cp·(T_in − T) + Q
Balance de Momento (Nivel)
Para nivel en tanque: la tasa de cambio de volumen = caudal entrada − caudal salida.
A·(dh/dt) = F_in − F_out → F_out = C_v·√h
Modelo de CSTR (Reactor)
Combinación de balance de masa reactivo + balance de energía. El término cinético r(C,T) introduce la no linealidad.
V·(dC_A/dt) = F·(C_A0−C_A) − V·r(C_A,T)
Modelos Lineales vs No Lineales
MODELO LINEAL
Principio de Superposición
Un sistema es lineal si satisface superposición: f(ax+by) = a·f(x) + b·f(y). Permite usar la Transformada de Laplace y todas las herramientas del control clásico. La gran mayoría de métodos de análisis y diseño suponen linealidad.
a_n·y^(n) + ... + a_1·ẏ + a_0·y = b_m·u^(m) + ... + b_0·u
  • Coeficientes constantes (parámetros no cambian con el estado)
  • Función de transferencia G(s) bien definida
  • Respuesta proporcional a la magnitud de la entrada
  • Herramientas: Laplace, Bode, Nyquist, Routh-Hurwitz
MODELO NO LINEAL
Linealización Alrededor del SS
Los procesos reales son no lineales (cinética de Arrhenius, raíz cuadrada en nivel, etc.). Para control clásico, se linealizan alrededor del punto de operación (estado estacionario) usando la expansión en serie de Taylor de primer orden.
f(x) ≈ f(x₀) + (∂f/∂x)|₀·(x−x₀) = f(x₀) + a·x'
  • Variables desviación: x' = x − x₀, y' = y − y₀
  • El modelo linealizado es válido solo cerca de x₀
  • Fuera del rango lineal: control no lineal (MPC, feedback linearization)
  • Ejemplo: √h ≈ √h₀ + (1/2√h₀)·h' = √h₀ + Kh·h'

Problema 1 — Modelado de nivel en tanque:
Un tanque cilíndrico de área transversal A = 2 m² tiene una válvula de salida con característica F_out = β·√h, donde β = 0.5 m^2.5/min. El caudal de entrada F_in es la variable manipulada. Punto de operación: h₀ = 4 m. Obtenga el modelo lineal en variables desviación y la función de transferencia.

▸ SOLUCIÓN — Linealización y Función de Transferencia
Modelo no lineal: A·(dh/dt) = F_in − β·√h
Estado estacionario (SS): 0 = F_in0 − β·√h₀ → F_in0 = 0.5×√4 = 1.0 m³/min
Linealizar F_out = β·√h:
F_out ≈ β·√h₀ + (β/2√h₀)·h' = 1.0 + (0.5/2×2)·h' = 1.0 + 0.125·h'
Modelo en variables desviación (h'=h−h₀, f'=F_in−F_in0):
A·(dh'/dt) = f' − (β/2√h₀)·h' = f' − 0.125·h'
2·(dh'/dt) + 0.125·h' = f'
Dividir entre 0.125: 16·(dh'/dt) + h' = 8·f'
Transformada de Laplace:
16s·H'(s) + H'(s) = 8·F'(s) → H'(s)·(16s+1) = 8·F'(s)
G(s) = H'(s)/F'(s) = 8 / (16s + 1) → Kp = 8 m/(m³/min), τ = 16 min
El proceso es de primer orden con ganancia 8 y constante de tiempo 16 min. Un escalón de 0.1 m³/min en F_in produce un cambio de 0.8 m en nivel en estado estacionario.

Función de Transferencia
// G(s) = Y(s)/U(s) · POLOS · CEROS · FORMA ESTÁNDAR
La función de transferencia G(s) es la relación entre la salida Y(s) y la entrada U(s) en el dominio de Laplace, bajo condiciones iniciales nulas. Codifica toda la dinámica del proceso en un cociente de polinomios en s. Sus raíces (polos y ceros) determinan completamente el comportamiento dinámico.
📐 Definición General
G(s) = Y(s)/U(s) = [b_m·s^m + ... + b_1·s + b_0] / [a_n·s^n + ... + a_1·s + a_0]
Forma factorizada:
G(s) = K · [(s−z₁)(s−z₂)...(s−z_m)] / [(s−p₁)(s−p₂)...(s−p_n)]
Condición de causalidad: n ≥ m (grado denominador ≥ numerador)
Ganancia estática Kp: Kp = G(0) = lim[s→0] G(s)
📐 Forma Estándar — Primer Orden
G(s) = Kp / (τs + 1)
Kp = ganancia estática [unidades de salida / unidades de entrada]
τ = constante de tiempo [min o s]
Con tiempo muerto (FOPDT):
G(s) = Kp · e^(−θs) / (τs + 1)
θ = tiempo muerto [min] — retardo puro de transporte
G(0) = Kp → ganancia en estado estacionario

▸ CASO REAL — INTERCAMBIADOR DE CALOR ECOPETROL BARRANCABERMEJA
Un intercambiador de carcasa y tubos en la refinería calienta nafta pesada con vapor a 4 bar. Prueba de lazo abierto: escalón del 10% en apertura de válvula de vapor → respuesta de temperatura de nafta. Datos identificados: Kp = 1.8°C/%apertura, τ = 6.5 min, θ = 1.8 min.

G(s) = 1.8·e^(−1.8s) / (6.5s + 1)

Ratio θ/τ = 1.8/6.5 = 0.28 → control moderadamente difícil. La Unidad II nos permite calcular ahora exactamente cuánto Kc máximo puede tener el controlador PID sin que el lazo oscile — esto es el margen de ganancia Gm, que veremos en la sección de Bode.
Sistemas de Primer Orden
// G(s) = Kp/(τs+1) · RESPUESTA TEMPORAL · ESCALÓN · RAMPA · IMPULSO
El sistema de primer orden es el bloque fundamental del modelado en procesos químicos. Un tanque de nivel, un intercambiador simple, un transmisor de temperatura — todos se comportan como sistemas de primer orden. Comprender su respuesta es esencial para interpretar pruebas de identificación y sintonizar controladores PID.
📐 Respuesta al Escalón Unitario (entrada: A·u(t))
G(s) = Kp / (τs + 1) → Y(s) = G(s)·(A/s) = Kp·A / [s·(τs+1)]
Respuesta temporal (antitransformada):
y(t) = Kp·A·(1 − e^(−t/τ)) para t ≥ 0
Valores clave:
t = τ → y = 0.632·Kp·A (63.2% del valor final)
t = 2τ → y = 0.865·Kp·A (86.5%)
t = 3τ → y = 0.950·Kp·A (95.0%)
t = 4τ → y = 0.982·Kp·A (98.2%) → prácticamente en SS
y(∞) = Kp·A ← valor final = ganancia × magnitud del escalón
📐 Respuesta al Impulso y a la Rampa
Al impulso δ(t) (U(s)=1):
Y(s) = Kp/(τs+1) → y(t) = (Kp/τ)·e^(−t/τ)
Pico en t=0: y(0) = Kp/τ. Decae exponencialmente.
A la rampa t·u(t) (U(s)=1/s²):
Y(s) = Kp/[s²·(τs+1)]
y(t) = Kp·[t − τ·(1 − e^(−t/τ))]
Para t≫τ: y(t) ≈ Kp·(t − τ) → sigue la rampa con retraso τ
Error de seguimiento de rampa = Kp·τ (offset permanente en vel.)
FIG. 3 — Respuesta escalón de sistema de primer orden. Características clave: en t=τ alcanza el 63.2% del valor final; la tangente en t=0 corta la asíntota exactamente en t=τ. El sistema alcanza el 98% en 4τ.

Problema 2 — Sistema de primer orden (Intercambiador Incauca):
Un intercambiador de calor en el Ingenio Incauca tiene G(s) = 1.5/(8s+1) donde la salida es temperatura de jugo (°C) y la entrada es apertura de válvula de vapor (%). Un operador aplica un escalón de 5% en la apertura. Calcule: (a) temperatura final en estado estacionario, (b) temperatura a t=8 min, (c) t para alcanzar 95% del valor final.

▸ SOLUCIÓN
(a) Valor final: y(∞) = Kp·A = 1.5 × 5 = 7.5°C (sobre el punto de operación)
(b) En t = τ = 8 min: y(8) = 7.5×(1−e^(−8/8)) = 7.5×(1−e^(−1)) = 7.5×0.632 = 4.74°C
(c) Para 95%: 0.95 = 1 − e^(−t/8) → e^(−t/8) = 0.05 → t = −8·ln(0.05)
t = −8×(−2.996) = 23.97 min ≈ 3τ
SS = +7.5°C · t(63%) = 8 min = τ · t(95%) ≈ 24 min = 3τ ✓

A. 6.00 unidades
B. 3.79 unidades
C. 5.00 unidades
D. 2.00 unidades

Sistemas de Segundo Orden
// G(s) = Kp·ωn² / (s²+2ζωns+ωn²) · AMORTIGUAMIENTO · SOBREIMPULSO
Los sistemas de segundo orden aparecen cuando se conectan en serie dos bloques de primer orden (CSTR con dos estados), o cuando el lazo cerrado de un controlador P con un proceso de primer orden produce una función de transferencia de segundo orden. El coeficiente de amortiguamiento ζ (zeta) es el parámetro más importante: determina si la respuesta oscila o no.
📐 Forma Estándar de Segundo Orden
G(s) = Kp·ωn² / (s² + 2ζ·ωn·s + ωn²)
Parámetros:
Kp = ganancia estática
ωn = frecuencia natural no amortiguada [rad/s]
ζ = coeficiente de amortiguamiento [adimensional]
Polos del sistema:
s₁,₂ = −ζ·ωn ± ωn·√(ζ²−1)
ωd = ωn·√(1−ζ²) ← frecuencia natural amortiguada
📐 Respuesta al Escalón (ζ < 1, subamortiguado)
y(t) = Kp·A·[1 − e^(−ζωnt)·(cos(ωd·t) + (ζ/√(1−ζ²))·sin(ωd·t))]
Sobreimpulso máximo OS:
OS% = 100·exp(−π·ζ/√(1−ζ²))
Tiempo de pico tp:
tp = π / ωd = π / (ωn·√(1−ζ²))
Tiempo de asentamiento ts (criterio ±2%):
ts ≈ 4 / (ζ·ωn) [criterio ±2%]
Clasificación por Coeficiente de Amortiguamiento
Subamortiguado (ζ < 1)
Polos complejos conjugados. Respuesta oscilatoria que decae. Más común en lazos de control diseñados con alta ganancia. El sobreimpulso aumenta al disminuir ζ.
p₁,₂ = −ζωn ± jωd (parte imaginaria ≠ 0)
Críticamente amortiguado (ζ = 1)
Polos reales iguales. Respuesta más rápida sin oscilación. Caso límite entre oscilatorio y no oscilatorio. Tiempo de asentamiento mínimo sin sobreimpulso.
p₁ = p₂ = −ωn (polo doble real)
Sobreamortiguado (ζ > 1)
Polos reales distintos. Respuesta monotónica sin oscilación pero más lenta que ζ=1. En la práctica: dos sistemas de primer orden en serie.
p₁,₂ = −ζωn ± ωn√(ζ²−1) (ambos reales negativos)
Marginalmente estable (ζ = 0)
Polos imaginarios puros. Oscilación sinusoidal sostenida — ni crece ni decae. El proceso oscila indefinidamente. En control: situación de máxima ganancia antes de inestabilidad.
p₁,₂ = ±jωn (Re(s) = 0)
FIG. 4 — Respuesta escalón para diferentes valores de ζ. ζ=0.707 (1/√2) minimiza el criterio ITAE y es el valor de diseño estándar en control de procesos. Mayor ζ → respuesta más lenta pero sin oscilación.

Problema 3 — Sistema de segundo orden CSTR linealizado:
El lazo de temperatura de un CSTR en la planta de NH₄NO₃ tiene la función de transferencia de lazo cerrado: G_LC(s) = 4 / (s² + 2s + 4). Calcule: (a) ωn, (b) ζ, (c) sobreimpulso OS%, (d) tiempo de pico tp y (e) clasifique la respuesta.

▸ SOLUCIÓN
Forma estándar: s² + 2ζωn·s + ωn² → comparar con s² + 2s + 4
(a) ωn² = 4 → ωn = 2 rad/s
(b) 2ζωn = 2 → ζ = 2/(2×2) = ζ = 0.5
(c) OS% = 100·exp(−π×0.5/√(1−0.25)) = 100·exp(−π×0.5/0.866)
OS% = 100·exp(−1.814) = 100×0.163 = 16.3%
(d) ωd = ωn·√(1−ζ²) = 2×√(0.75) = 2×0.866 = 1.732 rad/s
tp = π/ωd = π/1.732 = 1.81 s
ζ=0.5 → SUBAMORTIGUADO · OS=16.3% · tp=1.81 s. Aceptable para control, podría reducirse aumentando la ganancia derivativa.
❓ Un sistema de 2do orden tiene ωn=5 rad/s y ζ=0.707. ¿Cuál es el sobreimpulso aproximado?
A. 16.3%
B. 4.3%
C. 0% (sin sobreimpulso)
D. 73%
✅ OS% = 100·exp(−π×0.707/√(1−0.5)) = 100·exp(−π×0.707/0.707) = 100·exp(−π) = 100×0.0432 ≈ 4.3%. ζ=1/√2 es el valor óptimo ITAE para segundo orden.
Sistemas de Alto Orden y Aproximaciones
// POLOS DOMINANTES · REDUCCIÓN DE ORDEN · SISTEMAS CON CEROS · RESPUESTA INVERSA
Los procesos industriales reales tienen muchos estados (temperatura en cada plato de una columna, concentraciones en cada etapa de un tren de reactores) — son sistemas de orden muy alto. Para el diseño de controladores PID clásico usamos aproximaciones de orden reducido que capturan el comportamiento dominante del sistema sin perder información crítica.
Concepto de Polos Dominantes
POLO DOMINANTE
¿Qué polo "manda"?
En un sistema de alto orden, los polos más cercanos al eje imaginario son los dominantes: sus modos decaen más lentamente y determinan el comportamiento a largo plazo. Los polos más lejanos decaen tan rápido que su efecto es instantáneo — se aproximan como ganancias o como tiempo muerto adicional.
  • Polo dominante: |Re(p_dom)| ≪ |Re(p_rápido)|
  • Regla práctica: si |Re(p_rapido)| ≥ 5×|Re(p_dom)| → polo rápido no dominante
  • Los polos rápidos se "absorben" en un tiempo muerto equivalente
REDUCCIÓN DE ORDEN
Aproximación FOPDT
El método más usado en la industria: aproximar cualquier sistema de alto orden como un proceso de primer orden más tiempo muerto. Los polos no dominantes y los ceros de no mínima fase se incluyen en un θ equivalente.
G(s) ≈ Kp·e^(−θeq·s) / (τeq·s + 1)
  • Método de Skogestad: θ = θ_real + τ_2/2 + Σ(τ_i para i≥3)
  • Método de respuesta al escalón: identificar τ y θ gráficamente
  • Base de la sintonización IMC de la Unidad I
CEROS EN EL NUMERADOR
Sistemas de Fase No Mínima
Un cero en el semiplano derecho (Re(z)>0) causa que la respuesta temporal vaya inicialmente en la dirección opuesta antes de dirigirse al valor final. Este comportamiento se llama respuesta inversa y es especialmente problemático para el control.
  • Causa: dos caminos de causalidad con efectos opuestos
  • Ejemplo: columna de destilación — cambio de vapor causa primero bajada de concentración
  • Dificulta el control: el controlador "ve" la señal equivocada inicialmente
FIG. 5 — Respuesta en S de sistema de alto orden y su aproximación FOPDT. El punto de inflexión de la curva en S permite identificar gráficamente θ y τ para la sintonización PID (método Z-N lazo abierto).
📐 Método de Skogestad para Reducción de Orden
+
Para G(s) = Kp·e^(−θ₀s) / [(τ₁s+1)(τ₂s+1)(τ₃s+1)...], con τ₁ ≥ τ₂ ≥ τ₃...:

Paso 1: El polo dominante es τ₁ (el mayor). Este se convierte en τ_eq.
Paso 2: El segundo polo τ₂ se divide en dos partes: la mitad se suma al tiempo muerto, la otra mitad se descarta (para el cálculo IMC).
Paso 3: Todos los polos adicionales τ₃, τ₄... se suman íntegramente al tiempo muerto.

Resultado:
τ_eq = τ₁
θ_eq = θ₀ + τ₂/2 + τ₃ + τ₄ + ...

Ejemplo: G(s) = 2·e^(−0.5s)/[(10s+1)(3s+1)(1s+1)(0.5s+1)]
→ τ_eq = 10 min
→ θ_eq = 0.5 + 3/2 + 1 + 0.5 = 0.5 + 1.5 + 1 + 0.5 = 3.5 min
→ G_FOPDT(s) ≈ 2·e^(−3.5s)/(10s+1) → θ/τ = 3.5/10 = 0.35 → control moderado
▸ RESPUESTA INVERSA — COLUMNA DE DESTILACIÓN MANUELITA
En la columna de destilación del Ingenio Manuelita, al aumentar el vapor al rehervidor, la concentración de sacarosa en el destilado desciende inicialmente antes de subir. Esto ocurre porque el vapor primero arrastra más impurezas hacia la cabeza de la columna (efecto rápido) antes de mejorar la separación térmica (efecto lento dominante). El modelo tiene un cero en el semiplano derecho: G(s) = Kp·(1−τ_z·s) / [(τ₁s+1)(τ₂s+1)]. Un controlador PID estándar "confunde" la señal inicial inversa — por eso la acción derivativa D se desactiva en columnas con respuesta inversa.
Análisis de Estabilidad — Criterios Generales
// LAZO CERRADO · POLINOMIO CARACTERÍSTICO · CONDICIÓN BIBO · MARGENES
¿Por qué es crítica la estabilidad? Un lazo de control inestable puede causar oscilaciones que dañen equipos, generen condiciones peligrosas (temperatura fuera de rango en un reactor exotérmico) o produzcan producto fuera de especificación de forma irreversible. Nunca se debe operar un lazo que no haya sido verificado como estable.
Estabilidad BIBO
Bounded Input → Bounded Output. Un sistema es BIBO estable si para cualquier entrada acotada, la salida es acotada. Condición: todos los polos del lazo cerrado deben tener parte real estrictamente negativa.
Re(pᵢ) < 0 ∀ i → sistema BIBO estable
Función de Transferencia en Lazo Cerrado
Para el lazo estándar feedback negativo con controlador Gc(s) y proceso G(s):
T(s) = Gc(s)·G(s) / [1 + Gc(s)·G(s)·H(s)]
Polinomio Característico
El denominador de T(s): p(s) = 1 + Gc·G·H = 0. Las raíces de este polinomio son los polos del lazo cerrado. Si todas tienen parte real negativa → estabilidad.
p(s) = aₙsⁿ + aₙ₋₁sⁿ⁻¹ + ... + a₁s + a₀ = 0
Condición Necesaria de Estabilidad
Para que todas las raíces tengan parte real negativa, es necesario (pero no suficiente) que todos los coeficientes del polinomio característico sean del mismo signo (todos positivos para la convención estándar).
Si ∃ aᵢ ≤ 0 o aᵢ cambia de signo → sistema INESTABLE (sin calcular raíces)

Análisis en Frecuencia
// G(jω) · RESPUESTA SINUSOIDAL EN ESTADO ESTACIONARIO · MAGNITUD Y FASE
El análisis en frecuencia estudia cómo un sistema lineal amplifica o atenúa señales sinusoidales de diferente frecuencia, y qué desfase introduce entre la entrada y la salida. Sustituyendo s = jω en G(s) se obtiene G(jω), la respuesta en frecuencia. Esta información, representada en diagramas de Bode o Nyquist, permite determinar los márgenes de estabilidad sin necesidad de calcular los polos del lazo cerrado.
📐 Respuesta Sinusoidal en Estado Estacionario
Si u(t) = A·sin(ωt), entonces para t→∞:
y_ss(t) = A·|G(jω)|·sin(ωt + ∠G(jω))
Magnitud: |G(jω)| = amplitud de salida / amplitud de entrada
Fase: ∠G(jω) = desfase entre entrada y salida [rad o °]
Para G(s)=Kp/(τs+1), s=jω:
G(jω) = Kp/(jωτ+1) = Kp·(1−jωτ)/(1+ω²τ²)
|G(jω)| = Kp / √(1+ω²τ²)
∠G(jω) = −arctan(ωτ) [siempre negativo → atraso de fase]
📐 Frecuencias Características
Frecuencia de esquina (corner frequency):
ωc = 1/τ → |G(jωc)| = Kp/√2 ≈ 0.707·Kp (-3 dB)
∠G(jωc) = −45°
Frecuencia de cruce de ganancia (gain crossover):
ωgc: |G_LA(jωgc)| = 1 (0 dB)
Frecuencia de cruce de fase (phase crossover):
ωpc: ∠G_LA(jωpc) = −180°
Margen de ganancia Gm: frecuencia donde fase = −180°. Margen de fase Pm: fase cuando |G_LA|=1
Magnitud en dB
La magnitud en decibeles permite escala logarítmica y suma de módulos de bloques en serie en lugar de multiplicación. Rango industrial típico: −40 dB a +40 dB.
|G|_dB = 20·log₁₀(|G(jω)|) [dB]
Pendiente Asintótica
Cada polo de orden n añade −20n dB/década a la curva de magnitud para ω≫ωc. Cada cero añade +20 dB/década. Esta propiedad permite trazar Bode asintótico de forma manual.
n polos → −20n dB/dec | m ceros → +20m dB/dec
Tiempo Muerto θ en Frecuencia
El tiempo muerto e^(−θs) no afecta la magnitud del Bode pero añade una fase proporcional a la frecuencia. A altas frecuencias su efecto sobre la fase es devastador para la estabilidad.
∠e^(−jωθ) = −ω·θ [rad] = −ω·θ·(180/π) [°]
Función de Sensibilidad
S(s) = 1/[1+L(s)] donde L(s)=Gc·G es la función de lazo abierto. Cuantifica cómo las perturbaciones afectan la salida. El pico de |S(jω)| está relacionado con el margen de módulo Mm.
||S||∞ = 1/Mm → Mm_min = 0.5 (6 dB) para robustez
▸ ANÁLISIS EN FRECUENCIA — LAZO DE PRESIÓN PLANTA H₂SO₄
El lazo de presión del evaporador de la planta de ácido sulfúrico tiene G_LA(s) = Kc·2·e^(−0.3s)/(3s+1). A la frecuencia de cruce de fase ωpc donde ∠G_LA = −180°, se calcula: ωpc tal que −arctan(3ωpc) − 0.3·ωpc = −π. Por iteración: ωpc ≈ 4.7 rad/min. El margen de ganancia es Gm = 1/|G_LA(jωpc)| = (3s+1)/2 evaluado, lo que determina la Kc máxima antes de inestabilidad. Esto se visualiza en el diagrama de Bode de la sección siguiente.
Diagramas de Bode y Nyquist
// MAGNITUD dB · FASE ° · CURVA DE NYQUIST · CRITERIO DE NYQUIST
Los diagramas de Bode (magnitud y fase vs. log(ω)) y el diagrama de Nyquist (parte imaginaria vs. parte real de G_LA(jω)) son las dos representaciones gráficas estándar de la respuesta en frecuencia. Bode es más intuitivo para sintonización; Nyquist es más completo matemáticamente y permite analizar sistemas con tiempo muerto y ceros de fase no mínima.
FIG. 6 — Diagrama de Bode para proceso FOPDT. Línea continua azul: sin tiempo muerto. Línea punteada naranja: con tiempo muerto θ. El tiempo muerto añade fase negativa proporcional a ω, llevando el sistema antes a −180° y reduciendo el margen de ganancia Gm y el margen de fase Pm.
FIG. 7 — Diagrama de Nyquist. Criterio: el sistema en lazo cerrado es estable si la curva G_LA(jω) NO encircula el punto crítico (−1, 0j). Izquierda: sistema estable. Derecha: sistema inestable (encircula el punto crítico).
📐 Criterio de Nyquist Completo (Forma General)
+
El criterio de Nyquist formal: N = Z − P
N = número de encirculaciones de (−1,0) en sentido horario
Z = número de polos del lazo cerrado en el semiplano derecho (inestables)
P = número de polos del lazo abierto en el semiplano derecho

Para sistemas con P=0 (lazo abierto estable, usual en Ing. Química):
Z = N → si N = 0 (no encircula), Z = 0 → lazo cerrado estable ✓
Si N ≠ 0 → Z ≠ 0 → lazo cerrado inestable ✗

Caso especial con tiempo muerto: El tiempo muerto e^(−jωθ) hace que la curva de Nyquist espiralee al origen con una longitud de espiral que aumenta con θ. Para sistemas FOPDT, el criterio de estabilidad se aplica directamente verificando si la espiral pasa por dentro o fuera del punto (−1,0).
Márgenes de Estabilidad
// MARGEN DE GANANCIA Gm · MARGEN DE FASE Pm · MARGEN DE MÓDULO Mm
Los márgenes de estabilidad cuantifican cuánto puede variar la ganancia o la fase del lazo antes de que el sistema se vuelva inestable. En la práctica industrial, los modelos de proceso tienen incertidumbre — por eso se diseña con márgenes de seguridad adecuados que garanticen estabilidad robusta ante variaciones del proceso.
Margen de Ganancia — Gm
Factor por el cual la ganancia del lazo puede multiplicarse antes de que el sistema se vuelva marginalmente estable. Se calcula a la frecuencia de cruce de fase ωpc donde ∠G_LA = −180°.
Gm = 1/|G_LA(jωpc)| → Gm_dB = −20·log₁₀|G_LA(jωpc)|
Margen de Fase — Pm
Grados adicionales de fase negativa que puede soportar el lazo antes de ser inestable. Se calcula a la frecuencia de cruce de ganancia ωgc donde |G_LA| = 1 (0 dB).
Pm = 180° + ∠G_LA(jωgc) [positivo es buena señal]
Margen de Módulo — Mm
Distancia mínima de la curva de Nyquist G_LA(jω) al punto crítico (−1,0). Es el margen de estabilidad más conservador y completo: incluye simultáneamente variaciones de ganancia Y fase.
Mm = min_ω |G_LA(jω) − (−1)| = 1/||S||∞
Especificaciones de Diseño Recomendadas
MARGENMÍNIMO ACEPTABLERECOMENDADO (industria)EXCELENTEINTERPRETACIÓN
Gm (dB) 4–6 dB 6–12 dB > 12 dB La ganancia puede multiplicarse por Gm antes de oscilación. 6 dB = factor 2.
Pm (°) 20–30° 30–60° > 60° La fase puede retrasarse Pm grados antes de oscilación. 45°: buen balance velocidad/estabilidad.
Mm (dB) < −6 dB −6 dB ≤ −6 dB (Mm ≥ 0.5) Mm=0.5 equivale a pico de sensibilidad Ms=2. Mm<0.5 → sistema muy sensible a incertidumbre.

Problema 4 — Cálculo de márgenes de estabilidad:
El lazo de nivel del Tanque T-201 en la PTAR Cañaveralejo tiene la función de lazo abierto: G_LA(s) = Kc·2.5 / [s·(2s+1)]. Con Kc=1, calcule: (a) frecuencia de cruce de fase ωpc, (b) margen de ganancia Gm, (c) frecuencia de cruce de ganancia ωgc, (d) margen de fase Pm.

▸ SOLUCIÓN
G_LA(jω) = 2.5 / [jω·(2jω+1)] = 2.5 / [j2ω²·(−1) + jω]
= 2.5 / [−2ω² + jω] → parte real y imaginaria
(a) Frecuencia de cruce de fase ωpc (∠G_LA = −180°):
∠G_LA = −90° − arctan(2ω) = −180° → arctan(2ω) = 90° → 2ω → ∞
Esto ocurre solo para ω→∞ en este sistema (polo integrador). Para ωpc finita se necesita tiempo muerto o polos adicionales.
Con Kc=1: |G_LA(jω)| = 2.5 / [ω·√(1+4ω²)]
(c) Cruce de ganancia ωgc: |G_LA(jωgc)| = 1 → 2.5/[ωgc·√(1+4ωgc²)] = 1
Por prueba y error: ωgc ≈ 1.10 rad/min → ωgc = 1.10 rad/min
(d) Pm: ∠G_LA(j·1.10) = −90° − arctan(2×1.10) = −90° − arctan(2.20)
= −90° − 65.6° = −155.6°
Pm = 180° + (−155.6°) = 24.4°
Pm = 24.4° → margen marginal (mínimo recomendado 30°). Reducir Kc para mejorar el margen de fase. Sistema integrador: Gm → ∞ (teóricamente no se cruza −180° sin tiempo muerto).
❓ Un lazo tiene Pm=45° y Gm=10 dB. ¿Cuál afirmación es CORRECTA?
A. El sistema está en el límite de estabilidad
B. El sistema es inestable porque Pm < 60°
C. El sistema es estable con márgenes aceptables para la industria
D. Necesita más ganancia para ser estable
✅ Correcto. Pm=45° está dentro del rango recomendado (30–60°) y Gm=10 dB es bueno (mínimo 6 dB). El sistema es estable con márgenes adecuados. Pm=60° sería excelente pero no es requisito — Pm=45° es el estándar de diseño clásico para segundo orden con respuesta aceptable.

Criterio de Estabilidad de Routh-Hurwitz
// TABLA DE ROUTH · CONDICIÓN NECESARIA Y SUFICIENTE · GANANCIA CRÍTICA · ROUTH ESPECIAL
El criterio de Routh-Hurwitz permite determinar si todas las raíces de un polinomio tienen parte real negativa sin calcular las raíces explícitamente. Solo se requieren operaciones aritméticas sencillas sobre los coeficientes del polinomio característico. Es el método de análisis de estabilidad más usado en el diseño analítico de controladores PID en procesos químicos.
Construcción de la Tabla de Routh
📐 Algoritmo Paso a Paso
Polinomio característico: p(s) = aₙsⁿ + aₙ₋₁sⁿ⁻¹ + ... + a₁s + a₀
Paso 1 — Condición necesaria: Verificar que todos los coeficientes aᵢ sean positivos (mismo signo). Si no → inestable de inmediato.
Paso 2 — Construir tabla:
Fila sⁿ: aₙ aₙ₋₂ aₙ₋₄ ...
Fila sⁿ⁻¹: aₙ₋₁ aₙ₋₃ aₙ₋₅ ...
Fila sⁿ⁻²: b₁ b₂ b₃ ...
donde b₁ = (aₙ₋₁·aₙ₋₂ − aₙ·aₙ₋₃)/aₙ₋₁
b₂ = (aₙ₋₁·aₙ₋₄ − aₙ·aₙ₋₅)/aₙ₋₁
... continuar hasta fila s⁰
Paso 3 — Contar cambios de signo:
Número de raíces con Re>0 = número de cambios de signo en la 1ª columna de la tabla.

Problema 5 — Routh-Hurwitz para lazo de temperatura CSTR:
El polinomio característico del lazo cerrado de un CSTR con controlador PID es:
p(s) = s³ + 6s² + 11s + (6 + Kc)
Determine: (a) el rango de Kc para estabilidad, (b) la Kc crítica (ganancia límite), (c) la frecuencia de oscilación en la Kc crítica.

▸ SOLUCIÓN — Tabla de Routh para n=3
Coeficientes: a₃=1, a₂=6, a₁=11, a₀=(6+Kc)
Condición necesaria: Todos deben ser positivos → Kc > −6 ✓ (Kc positiva siempre cumple)
Tabla de Routh:
s³ │ 1 11
s² │ 6 (6+Kc)
s¹ │ b₁ 0
s⁰ │ (6+Kc)
Calcular b₁:
b₁ = (6×11 − 1×(6+Kc))/6 = (66 − 6 − Kc)/6 = (60 − Kc)/6
Condición de estabilidad (primera columna toda positiva):
a₂ = 6 > 0 ✓ (siempre)
b₁ = (60−Kc)/6 > 0 → Kc < 60
s⁰: (6+Kc) > 0 → Kc > −6 (siempre para Kc positiva)
Sistema estable para: −6 < Kc < 60 → para controladores Kc > 0: rango estable es 0 < Kc < 60
(b) Kc crítica = 60
(c) Frecuencia de oscilación en Kc=60:
En Kc crítica, fila s¹ = 0. La ecuación auxiliar es la fila s²:
6s² + (6+60) = 0 → 6s² + 66 = 0 → s² = −11
s = ±j√11 = ±j·3.317 rad/s
Kc_crit = 60 · ωosc = 3.317 rad/s = 0.528 Hz · Período de oscilación = 1.895 s
Casos Especiales en la Tabla de Routh
⚠️ Caso Especial 1 — Primer elemento de una fila = 0 (otros ≠ 0)
+
Si el primer elemento de una fila es cero pero el resto no, la tabla no puede continuar normalmente. Solución: sustituir el 0 por ε → 0⁺ (epsilon positivo pequeño) y continuar. Al evaluar el signo de la primera columna con ε→0, se determina si hay cambios de signo. Este caso indica que hay al menos un par de raíces conjugadas en el eje imaginario — el sistema está en el límite de estabilidad.
⚠️ Caso Especial 2 — Fila completa de ceros (raíces simétricas)
+
Ocurre cuando hay raíces simétricas respecto al origen: pares ±jω (eje imaginario) o pares ±σ (eje real) o cuartetos complejos. Solución: usar la ecuación auxiliar A(s), construida a partir de la fila anterior a la fila de ceros. Derivar A(s) respecto a s para obtener los coeficientes de la fila de sustitución y continuar la tabla.

Procedimiento:
1. Sea la fila anterior a la fila de ceros: a·sⁿ + b·sⁿ⁻² + c·sⁿ⁻⁴ + ...
2. Ecuación auxiliar A(s) = a·sⁿ + b·sⁿ⁻² + c·sⁿ⁻⁴
3. dA/ds = a·n·sⁿ⁻¹ + b·(n−2)·sⁿ⁻³ + ...
4. Los coeficientes de dA/ds reemplazan la fila de ceros.
Resumen Comparativo — Tres Criterios de Estabilidad
CRITERIOMÉTODOINFORMACIÓN QUE DALIMITACIÓNUSO PRINCIPAL
Routh-Hurwitz Tabla algebraica sobre coeficientes del polinomio característico Número de raíces inestables. Rango de Kc para estabilidad. Kc crítica y ωosc. Solo aplica a sistemas con coeficientes constantes. No funciona con tiempo muerto exacto. Diseño analítico de PID. Estabilidad paramétrica.
Margen de Ganancia Gm Diagrama de Bode / Nyquist a la frecuencia ωpc Factor de seguridad en ganancia. Cuánto puede aumentar Kc antes de inestabilidad. No captura variaciones simultáneas de ganancia y fase. Sintonización PID con Bode. Control industrial estándar.
Margen de Módulo Mm Distancia mínima en plano de Nyquist al punto (−1,0) Margen de estabilidad más completo. Considera variaciones simultáneas Gm y Pm. Más difícil de calcular analíticamente. Requiere función de sensibilidad S(s). Diseño robusto. Especificación de control avanzado.
❓ Para p(s) = s³ + 2s² + 4s + K, ¿cuál es el valor máximo de K para que el sistema sea estable?
A. K < 4
B. K < 8
C. K < 2
D. K < 16
✅ Tabla Routh: s³|1,4 s²|2,K s¹|b₁ s⁰|K.
b₁ = (2×4 − 1×K)/2 = (8−K)/2.
Condición: b₁ > 0 → K < 8. Y s⁰: K > 0.
Sistema estable para 0 < K < 8. Kc_crit = 8 → ωosc = √(a₀/b₁)|_(b₁→0) = √(K/b₁) → ω = √4 = 2 rad/s.
▸ APLICACIÓN INTEGRADA — PLANTA DE ÁCIDO NÍTRICO (ECOPETROL)
En la planta de ácido nítrico de Ecopetrol, el lazo de control de temperatura del reactor de oxidación catalítica tiene el polinomio característico en lazo cerrado:
p(s) = s⁴ + 7s³ + 14s² + (8+2Kc)s + Kc·τI⁻¹

El ingeniero de proceso aplica Routh-Hurwitz para encontrar el rango de Kc estable sin necesidad de calcular 4 raíces de un polinomio de cuarto grado. Adicionalmente, usa el diagrama de Bode (MATLAB margin()) para encontrar los márgenes Gm y Pm del lazo diseñado, verificando que Pm ≥ 45° y Gm ≥ 8 dB antes de poner el controlador en automático. Este proceso sistemático —Routh para límites analíticos + Bode para márgenes— es el estándar industrial para la puesta en marcha segura de lazos de control en plantas de proceso continuo.
